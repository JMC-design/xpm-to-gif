The XPM-TO-GIF dictionary


[Function]
xpm-file-to-canvas file-name color-table &key none-color => result


Takes a gif file name, color-table and a optional none-color. The none-color is the color you want to use for xpm colors set to None. A skippy canvas is returned.


[Function]
xpm-file-to-gif input-file output-file &key none-color => result


Takes the name of a gif input file, xpm output file and a optional none-color. The none-color is the color you want to use for xpm colors set to None.
